##  Project Overview

This report analyses Manchester City's performance using an interactive dashboard that captures match outcomes, goal statistics, and opposition dynamics. The objective of the analysis is to evaluate team effectiveness, identify strengths and weaknesses, and derive insights that support performance evaluation.

The dashboard was designed to provide a visual summary of:

- Match results
- Goal statistics
- Win/draw/loss rates
- Opposition breakdown

---

##  Executive Summary

Manchester City's statistical profile reveals strong competitive performance characterised by:

- High match participation
- Positive goal differential
- Strong win rate dominance

The team exhibits offensive efficiency while maintaining controlled defensive outcomes, resulting in a favourable goal difference.
